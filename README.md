# Корпоративная информационная система складского учета и аналитики

Корпоративная информационная система складского учета и аналитики, разработанная для автоматизации процессов учета товаров, поставок и анализа складских операций в АО «Новая мода».

## Возможности системы

* Учет товаров и категорий
* Контроль остатков на складе
* Приемка поставок
* Пополнение торгового зала
* Аналитика и отчетность
* Журналирование действий пользователей
* Разграничение прав доступа
* AI-помощник для работы с системой

## Используемые технологии

### Frontend

* React
* TypeScript
* Tailwind CSS

### Backend

* FastAPI
* Python

### Database

* PostgreSQL

## Архитектура системы

Система построена по клиент-серверной архитектуре:

* Frontend отвечает за пользовательский интерфейс;
* Backend реализует бизнес-логику и API;
* PostgreSQL используется для хранения данных.

## Основные модули

* Модуль управления товарами
* Модуль поставок
* Модуль аналитики
* Модуль авторизации
* Модуль AI-помощника
* Журнал операций

## Цель проекта

Автоматизация складского учета и повышение эффективности управления товарными запасами предприятия.

## Результаты разработки

В результате разработки была создана информационная система, позволяющая:

* сократить время обработки складских операций;
* повысить точность учета;
* улучшить контроль остатков;
* обеспечить централизованное хранение данных;
* получать аналитическую информацию для принятия решений.

## Перспективы развития

В дальнейшем система может быть расширена:

* мобильным приложением;
* интеграцией с ERP-системами;
* расширенной аналитикой;
* системой прогнозирования спроса.
